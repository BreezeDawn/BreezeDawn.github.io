第四步 - 更新静态文件，提交到 github 仓库

执行 $ npm install hexo-deployer-git --save - 安装关联 Github 的插件

执行 hexo d -g - 更新静态文件并提交到你的 github 仓库。

然后使用浏览器打开 https://username.github.io.git ，是不是可以外网访问了呢? 记住一定是https的哦，不然访问不到
